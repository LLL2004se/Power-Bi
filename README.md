# 🚀 Projects Included

## 1️⃣ Microsoft Visual Mart – Sales Dashboard

### 📖 Description

An interactive Power BI dashboard designed to analyze **sales performance, profit trends, and product insights** for a retail business.

### 🎯 Objectives

* Analyze sales and profit performance
* Identify top-selling products
* Track regional sales distribution

### 📊 Features

* Sales vs Profit analysis
* Category-wise performance
* Region-wise sales insights
* Time-based trends

### 🛠️ Tools Used

* Power BI
* Power Query
* DAX
